## Instruções

Faça o git clone na sua maquina e entre na pasta do projeto pelo terminal
#### É preciso ter instalado

1. O Ruby on Rails
2. O ImageMagick

Caso não tenha o ImageMagick, instale pelo comando:
```
sudo apt-get install imagemagick -y
```

## Rode o projeto

```
rake db:migrate
```

```
rails s
```
